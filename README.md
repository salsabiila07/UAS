# UAS TO-DO LIST
To-Do List merupakan aplikasi sederhana yang digunakan untuk mencatat dan mengelola daftar kegiatan atau tugas harian. Aplikasi ini membantu pengguna agar lebih terorganisir dengan menyediakan fitur penambahan, penampilan, penghapusan, serta penandaan tugas yang telah selesai
Alogoritma :
1. Mulai aplikasi.
2. Sistem menampilkan menu utama To-Do List.
3. Sistem memproses pilihan pengguna:
       - Menambah data tugas
       - Menampilkan daftar tugas
       - Menandai tugas selesai
       - Menghapus tugas
4. Sistem menampilkan hasil sesuai pilihan.
5. Jika pengguna memilih keluar, aplikasi selesai.
